Credit Scoring Model
Описание проекта

Проект посвящён построению модели кредитного скоринга — предсказанию вероятности дефолта клиента на основе его кредитной истории.

Данные представлены в событийном формате: одному клиенту соответствует несколько записей (кредиты), что требует агрегации признаков на уровне клиента.

Используемые технологии
Python
Pandas, NumPy
Scikit-learn
Matplotlib
Этапы работы
1. Загрузка данных
чтение parquet-файлов
объединение датасета
2. Анализ данных
исследование структуры
анализ типов признаков
3. Feature Engineering
агрегация событийных данных:
mean
sum
count
формирование признаков на уровне клиента
4. Обучение модели
разбиение на train/test
построение pipeline:
StandardScaler
Logistic Regression
5. Оценка качества
основная метрика: ROC-AUC
📈 Результаты

Модель Logistic Regression показала базовое качество, достаточное для baseline решения задачи кредитного скоринга.
